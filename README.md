# Angular - Angular Material Table With Sorting & Pagination
A basic application made using <b>Angular 8</b> and <b>Angular Material 8 </b>.  

## Version

| Name  | Version |
| ------------- | ------------- |
| Angular  | 9.0.3  |


## Project setup

#### Install dependencies
```
Navigate to root folder and type:
npm install
```
#### Run the server
```
npm start
```
```
ng serve -o
```

#### Run tests
```
ng test
```
