# Sensors Angular Application
A basic application made using <b>Angular 9</b> , <b>Angular Material</b> , <b>Leaflet maps</b> and <b>amCharts</b>.  

## Version

| Name  | Version |
| ------------- | ------------- |
| Angular  | 9.0.3  |


## Project setup

#### Install dependencies
```
Navigate to root folder and type:
npm install
```
#### Run the server
```
npm start
```
```
ng serve -o
```

#### Run tests
```
ng test
```
